# **CÁC BƯỚC LẤY VNCORENLP**

## **BƯỚC 1**

Truy cập vào đường link [Google Drive](https://drive.google.com/drive/folders/1YCUa_M_lObZ3XHOriPZLxd8jFlK3ktfr?usp=sharing).

## **BƯỚC 2**

Tải tất cả nội dung trong folder VnCoreNLP trên Google Drive vừa truy cập về máy.

## **BƯỚC 3**

Copy các nội dung vừa tải vào trong folder này.

Nếu có sẵn thì không cần thực hiện 3 Bước trên

## **BƯÓC 4**

Cài đặt [Java](https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html), tạo biến JAVA_HOME trong System Variables trong Environment Variables của máy tính, sử dụng đường dẫn tới folder jdk làm giá trị cho biến JAVA_HOME.
